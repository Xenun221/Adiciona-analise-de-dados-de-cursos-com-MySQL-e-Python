# Adiciona-analise-de-dados-de-cursos-com-MySQL-e-Python

Script de análise de dados extraídos de banco MySQL (plataforma_cursos_db).

Análises implementadas:
- Receita total por categoria de curso
- Média de valor por curso
- Distribuição de alunos por estado
- Filtro de alunos SP em andamento
- Análise temporal de matrículas (mês/dia)
- Identificação do melhor dia de vendas
- Taxa de conclusão do curso Python
- Análise de evasão (iniciantes)

Tecnologias: MySQL Connector, Pandas, Matplotlib
Tabelas utilizadas: alunos, matriculas (INNER JOIN)
